Master Formulas at a Glance

New Value After Increase

New Value = Original × (1 + r/100)
= Original × (100 + r) / 100
r = rate of increase (%)

New Value After Decrease

New Value = Original × (1 – r/100)
= Original × (100 – r) / 100
r = rate of decrease (%)

Finding Original Value

Original = New Value × 100 / (100 ± r)
Use + for increase, – for decrease
Reverse percentage technique

Successive % Change

Net % = a + b + (a×b)/100
(a = 1st change, b = 2nd change)
Use –ve sign for decrease

% Change Formula

% Change = (New – Old) / Old × 100
+ve = Increase, –ve = Decrease

Price–Quantity Inverse

If price ↑ by r%, then consumption
must ↓ by: r/(100+r) × 100 %
To keep expenditure constant
📈

Finding New Value After % Increase / Decrease

The most fundamental subtopic — directly applying the increase or decrease formula to find the resultant value. Appears in almost every SSC CGL paper.

💡

Golden Shortcut: For a r% increase, multiply by (100+r)/100. For a r% decrease, multiply by (100–r)/100. Never compute step-by-step — do it in one shot to save time!

Finding Original Value – Reverse Percentage

Given the value after a % change, work backwards to find the original. A high-frequency topic that trips many students.

💡

Key Trick: "After 20% increase the value is 600" → Original = 600 × 100/120. Always divide by the multiplier, not subtract the percent!

Successive / Compound Percentage Change

Two or more percentage changes applied one after another. Uses the net % formula to find overall effect in one step.

💡

Net % Formula: a + b + ab/100 (where a and b are % changes; use –ve for decrease). For 3 changes, apply the formula twice.

Percentage Change in Area, Perimeter & Volume

Apply % change concepts to geometric dimensions. Area depends on two dimensions — both changes multiply. A favourite in SSC CGL Tier I.

💡

Key Rule: For a rectangle, % change in Area = a + b + ab/100 (where a, b = % changes in length and breadth). For a square, area changes = 2r + r²/100. For a cube/sphere — volume involves 3 dimensions.
